支持的端点

端点方法认证描述
/v1/chat/completionsPOSTBearerOpenAI 原生格式
/v1/messagesPOSTBearerAnthropic 格式 → OpenAI
/v1/responsesPOSTBearerResponses 格式 → OpenAI
/v1/modelsGETBearer列出可用模型
/healthGET无公开健康检查

认证

所有 API 端点（/health 与 /stats 除外）都需要：

Authorization: Bearer sk-Aoki-MiMo

网关会自动注入后端 API key，客户端只需提供网关 token。

格式转换

Anthropic → OpenAI：转换 /v1/messages 请求（含系统提示、tool_use 块、流式）。

Responses → OpenAI：转换 /v1/responses（含 input 数组与函数调用）。

两种格式都会透明地转换为 /v1/chat/completions，并路由到最佳可用后端。
